# Country Explorer App - World Atlas

A modern React project to explore information about countries using the [REST Countries API](https://restcountries.com/).  
You can **search, filter by region, and view detailed country data** in an elegant and responsive UI.

---

## Features

- **Search** countries by name
- **Filter** countries by region
- View country details: population, capital, flags, and more
- Fast API fetching with **Axios**
- Error & Loading states handled gracefully
- Responsive design with modern UI
- Smooth navigation using **React Router**

---

## Tech Stack

- **React** (with hooks)
- **React Router v6**
- **Axios**
- **SCSS / CSS**
- **REST Countries API**
